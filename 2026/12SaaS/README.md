## 🛑 灵魂三问 (必须 Yes 才能开工)

1. **渠道**：已有流量/名单吗？(No = 不做)
2. **速度**：2 周能出 MVP 吗？(No = 不做)
3. **价值**：这一秒有人愿付费吗？(No = 不做)

---

## ⚖️ 铁律

- **3 个月生死线**：SaaS 项目上线 3 个月没到达预期付费目标 → **直接放弃**
- **不养闲鱼**：同时在研不超过 2 个项目
- **极度透明**：Build in Public
- **现金为王**：只相信已到账的钱
