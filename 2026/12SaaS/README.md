## 🔁 复盘节奏

### 周度 Review (每周日)
> **目的**：检查进度，快速纠偏

- [ ] 本周完成了什么？
- [ ] 主攻项目进度是否达标？
- [ ] 下周最重要的 3 件事是什么？
- [ ] 有没有需要放弃/暂停的事项？

### 月度复盘 (每月最后一天)
> **目的**：对照目标，调整资源分配

- [ ] 各项目 MRR / 用户数 vs 月度目标
- [ ] 投入产出比评估
- [ ] 下月资源分配是否需要调整？
- [ ] 是否有项目需要止损？

### 季度复盘 (Q1/Q2/Q3/Q4 末)
> **目的**：战略级调整，决定生死

- [ ] 3 个月生死线判断：未达标项目直接放弃
- [ ] 成功项目：加大投入
- [ ] 失败项目：复盘原因，提炼教训
- [ ] 下季度主攻项目重新排序
- [ ] 是否需要启动新项目/储备项目？
