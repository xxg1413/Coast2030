## 创意备选池



一.  Runnote 项目策划案（编程 + 创作者的大产品）

下面我直接给你一份可落地的策划案，你拿去就能开干。

------

## 1. 产品定位

**Runnote 是一个极简 AI IDE / 桌面工作台：用户只需要写文档（Markdown/Blocks），系统通过文档改动（diff）推导意图并执行动作（命令、生成、整理、发布），把结果自动回写沉淀为可复用资产。**

**核心范式：Doc → Diff → Do → Writeback**

------

## 2. 目标用户（从先锋用户切入）

### 第一批会付费的人

1. **AI 编程用户**：Claude Code / Cursor / Terminal 重度用户
2. **创作者**：写稿、脚本、封面、标题、分发、复盘
3. **独立开发者**：做产品同时做内容的人（你自己的画像）

### 后续扩展

1. 产品/运营：写方案→拉数据→生成报告
2. 研究党：实验记录→一键复现→结果沉淀

------

## 3. 竞争优势（你怎么赢 Antigravity / AI IDE）

你不拼“更强的 agent”，你拼：

- **更轻**：像 Typora 一样写作体验
- **更稳**：默认人确认、默认安全白名单
- **更可沉淀**：执行结果自动回写，越用越值钱
- **更跨场景**：代码/内容/任务统一在一份“可执行文档”里

------

## 4. Wow Moment（必须打穿）

> 用户只改了文档，保存后右侧自动完成该做的事，并把结果写回文档。

### 编程 Wow

- 勾选任务 ✅ “Add tests”
- 自动运行 `test`
- 失败自动给出修复 patch（先 diff 再应用）
- 结果回写：通过/失败原因/下一步

### 创作者 Wow

- 改标题方向 + 发布平台
- 自动生成：10 标题 + 3 版文案 + hashtags
- 自动生成分发计划
- 输出回写到 `## Outputs`

------

## 5. 核心机制设计（文档驱动执行）

### 5.1 Diff-as-Intent（用改动表达意图）

每次保存文档：

- 系统拿到 `previous_doc` vs `current_doc` 的 diff
- 只让模型读 diff（更稳定、更便宜、更不跑偏）
- 输出一个 **Execution Plan**

### 5.2 执行器（Do Engine）

支持 3 类动作：

1. **Terminal Actions**：跑命令、启动服务、跑测试
2. **File Actions**：改文件（必须先展示 diff 才能 apply）
3. **Creator Actions**：生成标题/脚本/拆条/分发清单

### 5.3 Writeback（沉淀回写）

统一回写到文档固定区块：

- `## Results (auto)`：执行结果摘要
- `## Outputs (auto)`：生成内容/产物链接/文件清单
- `## Next (auto)`：下一步建议/按钮

------

## 6. 安全策略（你的产品信任来源）

默认三道闸：

1. **默认不自动执行**：先生成 Plan
2. **默认白名单**：只允许 test/lint/build/dev 等安全命令
3. **默认沙箱**：只能在 workspace 内读写

危险命令强制确认（rm/sudo/curl|sh 等）

> 这会让你比“全自动 agent IDE”更让人敢用。

------

## 7. 产品形态（桌面端 UI）

推荐三栏（最强、最清晰）：

- 左：Workspace（项目/资料库/模板）
- 中：Runnote 文档（Markdown/Blocks）
- 右：Do Panel（Terminal + Agent + Actions）

底部：Timeline（执行历史、回滚点）

------

## 8. Roadmap（7天 / 30天 / 90天）

### MVP（7 天可发）

- Markdown 编辑器（Typora 手感）
- 内置终端（多 tab）
- 文档版本快照（diff）
- Diff → Plan（生成执行计划）
- 手动确认执行命令
- 执行结果自动回写到文档

✅ 验收：完成一次真实项目迭代/一次内容分发闭环

------

### v1（30 天：可收费、可口碑）

- Actions 面板：从文档提取“可执行按钮”
- 自动总结（日志压缩成结论）
- 文件修改：Diff → Apply
- Run Profiles：Dev / Creator 两套预设
- 模板库：一键创建“项目 Runbook / 内容 Runbook”

------

### v2（90 天：平台化）

- 模板市场（Creator Pack / Dev Pack）
- 连接器：GitHub、X、YouTube、Notion（先做你最常用的）
- Team 版：共享模板、权限、审计
- 云同步：配置/模板/记忆同步（续费引擎）

------

## 9. 盈利模式（投机 & 投资）

### 投机（快钱）

1. **Lifetime 限量**：$79～$129
2. **Pro 订阅**：$12～$19/月（解锁自动总结、回写、Agent改文件）
3. **模板包**：$19～$49（Creator Pack / Dev Pack）

### 投资（长线）

1. **Team**：$25/席位/月（共享 Runbook + 审计）
2. **Marketplace 抽成**：20%～30%
3. **Enterprise**：私有模型/安全策略/本地推理
4. **云同步**：最强续费理由
