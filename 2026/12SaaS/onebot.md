# OneBot - 极简AI聊天机器人平台

> **核心定位**：一键部署的AI聊天机器人解决方案
> **域名**：onebot.ai  
> **当前访问量**：1/月（新域名）

---

## 🎯 产品定位

### 一句话描述
"一个配置文件，部署你的专属AI助手"

### 目标用户
- **中小企业**：需要客服机器人但预算有限
- **个人开发者**：想要快速给网站添加AI助手
- **内容创作者**：需要粉丝互动机器人

---

## 💡 核心功能

### MVP功能（第一阶段）
1. **配置化部署**
   - YAML/JSON配置文件定义机器人行为
   - 无代码部署，一键上线

2. **多平台集成**
   - 网页聊天组件
   - Telegram Bot
   - Discord Bot
   - 微信公众号（后期）

3. **知识库管理**
   - 上传文档自动构建知识库
   - RAG（检索增强生成）

### 第二阶段功能
- 对话分析仪表盘
- 多轮对话流程编排
- 自定义函数调用（Function Calling）
- API访问

---

## 🎨 技术栈

```
Frontend:  Next.js + shadcn/ui
Backend:   Cloudflare Workers + D1
AI:        OpenAI / DeepSeek API
Vector DB: Pinecone / Cloudflare Vectorize
Deploy:    Cloudflare Pages
```

## 🎯 差异化优势

### vs Botpress
- ✅ 更轻量，配置化优先
- ✅ 更便宜，适合小团队
- ❌ 功能相对简单

### vs Voiceflow
- ✅ 开发者友好（代码/配置）
- ✅ 自托管选项
- ❌ UI不够可视化

### 我们的定位
**"技术人员的最爱，5分钟部署完成的AI助手"**
