## 🔁 复盘节奏
- **月度复盘**：对照季度目标与当月现金流，判断是否需要加速/止损。
- **季度复盘**：围绕执行节奏表进行偏差分析，必要时调整项目组合。
- **年度复盘**：评估是否达成结余目标，确定下一年主题与资源分配。
