## 📅 周复盘 (Weekly Review)
**时间**：每周日晚 20:00 (15分钟)
**核心**：只关注“增量”和“下周重点”

### 1. 关键指标 (Metrics)
| 维度 | 指标 | 本周数值 | 变化 |
| :--- | :--- | :--- | :--- |
| **💰 现金** | **总到账金额** | $____ | 🟢/🔴 |
| **🕵️ Hunter** | 提交/接单数量 | ____ | - |
| **🧑‍🌾 Farmer** | SaaS MRR | $____ | - |
| **📣 流量** | 涨粉总数 | ____ | - |

### 2. 🚦 红绿灯 (Stop/Start/Continue)
*   **🔴 Stop (别做了)**：本周哪件事浪费了时间且没产出？
*   **🟢 Start (新尝试)**：下周想尝试的一个新增长点？
*   **🟡 Continue (保持)**：做对了什么？

### 3. I am Blocked (卡点)
*   *当前最大的阻碍是什么？需要买工具/外包/砍需求吗？*

### 4. 下周 "Big 3"

3.  *(第三件事)*

## 🔭 季度战略会 (Quarterly Strategy)
**时间**：每季度末 (Q1/Q2/Q3/Q4)
**核心**：调整双轨策略比重

### 1. 季度目标校准
*   **上一季度评分**：(0-10分)
*   **主要偏差原因**：是没执行到位？还是方向错了？

### 2. Hunter vs Farmer 配比
*   当前时间/精力分配：[Hunter __% / Farmer __%]
*   **下季度调整**：
    *   *现金不足？* -> **增加 Hunter 比重** (接外包/挖洞)
    *   *现金充裕？* -> **增加 Farmer 比重** (做产品/写内容)

### 3. 机会雷达
*   复盘 Idea Backlog，是否有新的风口需要加入 Pipeline？
*   是否有旧的“僵尸项目”占用了资源？
