## Cloudflare 优先策略

以下场景优先使用 Cloudflare 服务：
- **静态网站托管**: Cloudflare Pages
- **Serverless API**: Cloudflare Workers
- **数据库**: Cloudflare D1 (SQLite) / KV Storage
- **文件存储**: Cloudflare R2
- **边缘计算**: Cloudflare Workers

以下场景可能需要独立服务器：
- 需要长时间运行的后台任务
- 需要 GPU 计算的 AI 服务
- 需要特定系统环境的服务
- 数据合规性要求（如需中国大陆服务器）
