## 🔥 增长策略

### Phase 1: 冷启动 (0 → 1k 订阅)
- **Shorts 暴力拉流**：每天发 1 条，展示项目高亮时刻
- **首发长视频**："I built an emotional AI CLI in 24 hours"
- **SEO 优化**：标题/描述/标签 针对 "AI coding" "indie developer"

### Phase 2: 突破 YPP (1k → 5k 订阅)
- **系列化内容**：Wukong CLI 开发全记录（5-10 集）
- **跨平台引流**：X/Reddit 发预告 + 链接
- **合作推广**：找 AI 开发领域同行互推

### Phase 3: 复利期 (5k → 1w 订阅)
- **直播 Coding**：24h Live Coding 冲击观看时长
- **社区建设**：Discord 会员群
- **产品联动**：视频内推广 SaaS 产品

## 🛠 制作流程

### 设备清单
- [x] 屏幕录制：OBS / ScreenFlow
- [x] 剪辑：DaVinci Resolve / Final Cut Pro
- [ ] 麦克风：需优化
- [ ] 摄像头：可选（屏幕录制为主）

### 视频制作 SOP
1. **选题**：对齐内容支柱 + SEO 关键词
2. **大纲**：5 分钟写脚本框架
3. **录制**：边 Coding 边解说
4. **剪辑**：去除冗余 + 加字幕 + 配乐
5. **SEO**：标题/描述/标签 优化
6. **缩略图**：醒目 + 文字 + 面部表情
7. **发布**：最佳时间 + 社交媒体同步推广
