### 3. 安装步骤 (1:30-4:00)

#### Step 1: 安装Claude Code (1:30-2:30)

**字幕脚本**:
```
[1:30] "Step 1: Install Claude Code globally"
[1:40] "npm install -g @anthropic-ai/claude-code"
[2:00] "Wait for installation..."
[2:20] "Installation complete ✅"
```

**屏幕操作**:
1. 显示完整命令（大字）
2. 逐字输入（正常速度）
3. 按Enter
4. 显示下载进度（2倍速）
5. 显示安装成功消息

**视觉元素**:
- 命令卡片：黑色背景，绿色文字
- 进度条：底部显示百分比
- 成功动画：✅ + 闪光效果

#### Step 2: 验证安装 (2:30-3:15)

**字幕脚本**:
```
[2:30] "Step 2: Verify installation"
[2:40] "claude --version"
[2:55] "Version displayed = Success ✅"
```

**屏幕操作**:
1. 输入 `claude --version`
2. 显示版本号
3. 输入 `claude --help`
4. 快速滚动help输出（高亮关键命令）

**视觉元素**:
- 版本号放大显示
- 关键命令高亮：黄色标记
- 对勾动画：✅

#### Step 3: 首次启动 (3:15-4:00)

**字幕脚本**:

3. 显示提示符（等待输入状态）
4. 输入 `exit` 退出

**视觉元素**:

- 提示符闪烁动画
