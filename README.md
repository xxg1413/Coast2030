# Coast2030


这是Pxiaoer的5年退休计划，时间段从2026年1月1日到2030年12月31日，目标是实现财务自由，提前退休。

## 核心目标

- 2026年：总收入300万，结余100万
- 2027年：总收入400万，结余150万
- 2028年：总收入800万，结余250万
- 2029年：总收入1500万，结余600万
- 2030年：总收入2500万，结余900万


## 核心策略

2026年：
- SaaS创业：175万 -> [12SaaS](./12SaaS/README.md)
- 漏洞挖掘：100万 -> [漏洞挖掘](./漏洞挖掘/README.md)
- 自媒体：25万 -> [自媒体](./自媒体/README.md)


结余：100万
