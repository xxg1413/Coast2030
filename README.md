## 🔁 复盘节奏
- **月度复盘**：对照年度目标与当月现金流，判断是否需要加速/止损。
- **季度复盘**：检查项目组合、投入产出与主线偏移，必要时调整双轨策略。
- **年度复盘**：评估是否达成当年结余目标，重置下一年主题与资源分配。



## 🛑 铁律 (违反即死)

1.  **3个月生死线**：SaaS项目上线3个月没有到达预期付费目标 → **直接放弃**。
2.  **不养闲鱼**：2026 年最多尝试 12 个项目，同时在研不超过 2 个。
3.  **极度透明**：Build in Public，护城河是“节奏”，不是代码。
4.  **现金为王**：只相信已到账的钱。
