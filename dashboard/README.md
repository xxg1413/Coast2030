## Coast 2030 Dashboard

这是一个 Next.js 项目，用于管理 Coast 2030 的财务数据。

### 运行

```bash
pnpm dev
```


## 功能 (v0.3)

### 📊 核心仪表盘
- **年度看板**: 实时追踪 2026 年度目标、现金流、MRR 和 Hunter 收入。
- **本周焦点**: 管理每周核心任务，支持添加、完成状态切换。
- **本月里程碑**: 追踪月度关键战役，支持添加、删除 (ID-based) 和完成。

### 💰 财务明细 (新)
- **月度账本**: 查看每月详细收入流水 (按月份筛选)。
- **记账功能**: 快速记录收入 (类型、项目、金额、备注)。
- **数据可视化**: 自动计算当月总收入，不同收入类型高亮显示。

### 🛠️ 技术特性
- **数据持久化**: 
    - 开发环境: `sql.js` (SQLite) + 本地文件持久化，支持热重载。
    - 生产环境: Cloudflare D1 (SQLite) 边缘数据库。
- **移动端适配**: 响应式布局，支持手机端完美访问 (表格横向滚动、按钮触控优化)。
- **安全认证**: 自定义登录页面 + Cookie 认证。
